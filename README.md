# LED-PUSHBUTTOM
Neste projeto, que foi realizado no Wokwi, é funciona simplismente dessa forma: Quando o pushbuttom é pressionado o LED vermelho é ligado instântaneamente, mas é desligado assim que o pushbuttom para de ser pressionado. Foi ultilizado: 1 LED, 1 Resistor, 1 Half breadboard e 1 Pushubuttom.
